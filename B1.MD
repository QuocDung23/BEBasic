Key Words:

- Client - Server.
- HTTP - HTTPS: + Request.
                + Respond.
                + Method.
- API.
- SSR - CSR.

SSR: + Server Side Rendering, render từng ít, server load
CSR: + Client Side Rendering, render all html, máy chủ load



HTTP:
- Request: + URL
           + Status code:
           + Method: - Post: đăng lên
                     - Get: lấy(đọc) data
                     - put: update tất cả
                     - patch: update 1 số thứ cần
                     - delete: xóa

AIP: lấy info trong server cần thiết ra.
